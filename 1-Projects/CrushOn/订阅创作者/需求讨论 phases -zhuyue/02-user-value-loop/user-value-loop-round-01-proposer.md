## 1. 用户订阅决策路径拆解（修正版）

### 1. 看到创作者页
- 用户心里的问题：这个创作者“订阅后到底多看什么”，是一次性图包，还是持续更新的内容宇宙？
- PRD 当前答案：只有创作者页入口，且仅显示“可订阅”；对“订阅卖什么”解释极弱。与 C1.2 一致地回避了聊天权益，但没有把“持续访问权”讲清。
- Pixiv FANBOX / DLsite 做法：创作者页会先展示更新频率、历史连载、样张、文章体裁，让用户先判断这是“持续供给”还是“单次买断”。
- 漏斗损耗预估：25%-35%。当前入口能触发点击，但无法建立明确价值预期。

### 2. 价格评估
- 用户心里的问题：这个价格属于“试水”还是“铁粉支持”？值什么内容量？
- PRD 当前答案：只看到该创作者选择的一档价格，缺少档位语义、缺少对应权益说明。
- 对标做法：FANBOX 的价格档通常天然绑定更新密度、内容类型、支持意味；DLsite 则靠单品页清楚交代内容规格。
- 漏斗损耗预估：15%-25%。用户不是嫌贵，而是不知道为什么是这个价。

### 3. 专属内容预览（模糊态）
- 用户心里的问题：模糊下面到底是什么量级？是 3 张图还是 30 张图？有剧情吗？
- PRD 当前答案：未订阅高斯模糊，但没有明确“共多少组、多久更新一次、最近一次更新时间”。
- 对标做法：至少会露出标题、发布日期、部分缩略、正文摘要、系列归属。
- 漏斗损耗预估：20%-30%。纯模糊能制造好奇，但不能完成价值确认，尤其不利于 C1.5 所需的“世界观档案+剧情连载”。

### 4. 支付前价值确认
- 用户心里的问题：订阅是否包含聊天？是否自动续费？取消是否方便？NSFW 在我设备上能否正常消费？
- PRD 当前答案：支付主体明确，但“订阅不含聊天权益”虽是关键事实，却不构成强提示；取消路径隐蔽；到期/续费说明弱。
- 对标做法：购买前通常会明确“你将获得什么、不包含什么、多久续费一次、如何取消”。
- 漏斗损耗预估：10%-20%。这里的损耗不只影响首购，更影响退款/投诉/次月续费。

### 5. 支付完成
- 用户心里的问题：我现在立刻解锁了什么？
- PRD 当前答案：成功邮件；站内即时引导未明确。
- 对标做法：支付成功页直接带去“本月新解锁内容清单”。
- 漏斗损耗预估：5%-10%。不是支付失败，而是支付后的即时满足感不足。

### 6. 首次内容消费
- 用户心里的问题：首日能否一口气消费到“值回票价”？
- PRD 当前答案：可见付费相册、OC 对话图；但无世界观文稿、连载、投票、订阅者评论区。
- 对标做法：FANBOX 的首日消费通常至少包括历史贴文、设定稿、世界观说明、月更承诺。
- 漏斗损耗预估：30%-45%。这是当前最大断点，直接决定 C1.4 的首月续费能否接近 40%-45%。

### 7. 7 日回访
- 用户心里的问题：这一周有新东西吗？我为什么回来？
- PRD 当前答案：只有被动等用户自己回来；无 push、无更新提醒、无内容节奏承诺。
- 对标做法：更新通知、作者帖文、评论互动会主动把人拉回。
- 漏斗损耗预估：35%-50%。

### 8. 30 日复看
- 用户心里的问题：这个月我到底消费了多少、还有什么值得续？
- PRD 当前答案：只有订阅管理页；无“本月已解锁总结”、无内容里程碑。
- 对标做法：用户能看到自己这一月解锁了哪些贴文/系列，形成沉没价值。
- 漏斗损耗预估：20%-35%。

### 9. 续费决策
- 用户心里的问题：下个月会继续给吗？退订会失去什么？
- PRD 当前答案：Auto Renew 有状态展示，但缺少内容承诺、提醒、身份权、参与感。
- 对标做法：续费不是靠开关，而是靠持续更新预告和订阅身份。
- 漏斗损耗预估：若维持现状，首月续费更接近 20%-30%，低于 C1.4 底线。

## 2. 内容付费墙的“首月价值感知”设计评估

按当前 PRD，首日用户能看到的专属内容规模大致会是：

- 头部创作者：10-30 组相册，5-15 个对话专属图片节点。若历史库存足，$4.99 可成立，$14.99 勉强，$29.99/$49.99 很危险。
- 腰部创作者：4-10 组相册，2-6 个对话图节点。只够支撑 $4.99 的试水心智。
- 长尾创作者：0-3 组相册，0-2 个对话图节点。很容易退化成“花钱开一个空柜子”。

对比 FANBOX ¥500 档，用户通常默认月更 2-4 组图、1-2 篇设定/世界观/开发日志；其核心不是纯图片数量，而是“持续更新的作者存在感”。Crushon 当前能给到的是“付费相册+对话图”，缺文字档案、缺连载、缺订阅者互动，内容密度低于同价位参考物。

四档价值匹配度判断：
- `$4.99`：仅当至少有“历史库存 + 月更承诺”时成立。
- `$14.99`：需要明确月更节奏、系列化内容、世界观档案，否则溢价无锚点。
- `$29.99`：需要明显身份权或高频连载，目前完全不匹配。
- `$49.99`：接近收藏/赞助层，需要强身份权与高稀缺内容，当前 PRD 不具备。

结论：如果内容形态只有“付费相册 + 对话图”，按 C1.5 远不足以支撑 40%-45% 首月续费，最多支撑首购 curiosity，不足以支撑次月 habit。

## 3. 订阅的“访问权”与“身份权”缺口分析

### 访问权差距
缺失的关键访问权要素：

- 世界观档案入口：直接影响 C1.5，是把“图包”升级为“持续访问权”的核心，优先级 P0。
- 创作者文字更新/blog：直接提升 7 日回访与 30 日复看，优先级 P0。
- 连载目录/系列归档：让用户知道自己在追什么，对 C1.4 续费有中高影响，优先级 P1。
- 更新提醒/站内 push：对首月续费影响极大，优先级 P0。
- 订阅者评论区：形成轻互动与内容讨论，对留存有中等影响，优先级 P1。
- 投票入口：把用户从被动观看者变成参与者，对 C1.5 稀缺性与 C1.2 互补关系都有帮助，优先级 P1。

### 身份权差距
当前订阅者在站内基本不可见。没有徽章、没有累计订阅时长、没有订阅者社群、没有评论权，等于没有身份资产。这样一来，退订几乎零心理成本，订阅就会更像一次性购买，反向削弱 C1.2 所要求的“订阅卖持续访问权、打赏卖情绪表达”的分工。身份权不是装饰，它决定用户为什么不轻易退。

## 4. 四档定价 + 创作者四选一的心智评估

四档用户心智大致应是：

- `$4.99`：轻度试水
- `$14.99`：铁粉持续追更
- `$29.99`：高支持者
- `$49.99`：收藏/赞助型

问题在于，PRD 让创作者四选一，但用户在不同创作者页看到的只是“一个价格”，没有统一的阶梯权益定义。结果不是“分层定价”，而是“随机标价”。这会形成价格迷惑：同样是 AI 创作者，为什么这里 `$4.99`，那里 `$29.99`，用户无法判断贵在哪。

缺失的核心是 Patreon tier 式阶梯权益。当前不是多层权益，只是多档价格。没有“不同价格解锁不同内容/身份/参与权”，高价位缺乏认知锚点。

90 天切档冷却 + 切档中断所有 Auto Renew 也偏重创作者端管理，轻视老订阅者体验：
- 对老订阅者伤害：续费关系被强制打断，增加流失。
- 对创作者试错压制：不敢频繁调价，最终更可能保守选低价或拍脑袋选高价。
- 对用户心智伤害：价格不稳定会削弱信任。

在已确认“订阅和会员价无关”的前提下，撞会员价的问题可以跳过，但“单档随机价 + 无阶梯权益”仍是定价层面的结构性漏洞。

## 5. 续费触发机制缺口

对 40%-45% 首月续费阈值的影响判断：

- 到期前提醒：高影响，少 5-8 个点，P0。
- 创作者新内容主动 push：高影响，少 8-12 个点，P0。
- 订阅者身份感：中高影响，少 4-7 个点，P0/P1 边界，建议 P0。
- 月度“已解锁”消费总结：中影响，少 3-5 个点，P1。
- 创作者连载承诺：高影响，少 6-10 个点，P0。
- 订阅者投票参与：中影响，少 3-6 个点，P1。

按 FANBOX 类比的最低必备要素，应至少包含：
- 更新提醒
- 创作者文字更新/连载承诺
- 世界观或系列归档入口
- 到期前提醒

没有这四项，订阅基本不是“月订阅”，而是“带自动续费的一次性图包购买”。

## 6. 用户认知风险：订阅无聊天权益是否会造成抗议

会，而且概率不低。很多用户对“订阅”天然预期是“更深互动”或“聊天权限增强”。Crushon 又是强聊天语境产品，所以若不前置说明，用户极易把“订阅创作者”误解为“订阅聊天关系”。

当前最大风险不是用户不能接受纯内容订阅，而是用户在支付前没被明确教育。支付后再发现“聊天还要会员+credits”，会产生明显挫败感，并直接伤害 C1.1 的增量钱包率，因为用户会把后续消费视为重复收费。

PRD 必补的价值澄清机制：
- 入口旁固定文案：`Unlock subscriber-only albums and story content. Chat is not included.`
- 支付前确认框：明确“本订阅不包含聊天权益、会员权益、credits”。
- 购买成功页新手引导：第一屏直接带去“你刚解锁的专属内容”，避免用户先去聊天再撞墙。
- 模糊态预览页增加内容结构说明：相册数、系列数、最近更新时间。

## 7. iOS 端 NSFW 订阅的用户体验风险（用户要求同步上线）

### 方案 A：iOS 完全隐藏 NSFW 订阅入口
- 用户体验代价：最高。用户会认为创作者没开订阅或自己账号异常。
- 转化损耗：60%-80%。
- 合规风险：最低。
- 判断：最安全，但会制造大量客服与信任问题。

### 方案 B：展示但跳 Web 成交
- 用户体验代价：中等。路径长，但用户至少知道有订阅。
- 转化损耗：35%-55%。
- 合规风险：最高，若展示过强购买引导，审核风险显著。
- 判断：商业上最好，合规上最危险，C1.6 下不应直接作为默认。

### 方案 C：iOS 展示关注/预览，订阅按钮模糊但可点击跳 Web
- 用户体验代价：中低。至少完成认知教育，且保留迁移路径。
- 转化损耗：40%-60%。
- 合规风险：中等，取决于文案与跳转方式。
- 判断：三者里最平衡，更接近可执行折中。

若 C1.6 是 BLOCKER，用户侧建议优先 C，再配极弱交易文案与外部完成支付。

## 8. 用户侧漏洞清单

1. 订阅入口只显示“可订阅”，未显示内容结构与更新频率。影响 C1.1/C1.4/C1.5。应补“历史内容量+最近更新时间+月更承诺”。
2. 模糊预览只有遮罩，没有标题/摘要/数量。影响 C1.4/C1.5。应补系列名、张数、发布日期。
3. 支付前未强提示“订阅不含聊天权益”。影响 C1.1/C1.2/A6。应加固定澄清文案与二次确认。
4. 支付成功后缺站内即时导流。影响 C1.4。应跳转“本次已解锁内容页”。
5. 订阅管理只有状态，没有消费总结。影响 C1.4。应补“本月解锁了什么”。
6. 取消订阅入口放在创作者页，不在订阅列表。影响 C1.1，易引发投诉。应在列表中提供管理入口。
7. 没有到期提醒/更新提醒。影响 C1.4。应列为 P0。
8. 四档价格无统一权益映射。影响 C1.1/C1.4。应建立平台级价格心智锚点。
9. 90 天切档冷却且中断 Auto Renew。影响 C1.4。应至少保护存量订阅者不受新价波动冲击。
10. 内容形态只有相册和对话图。影响 C1.2/C1.5。应尽快补世界观档案、连载、投票。

## 9. Phase 2 评分

- 订阅是否带来钱包外延：`1/2`
理由：有机会形成新增付费入口，且不与会员直接撞价；但“聊天不含”若不讲清，会拖累 C1.1 增量钱包率。
- 内容付费墙续费力：`0/2`
理由：当前更像图包订阅，缺更新机制、身份机制、连载机制，难达 C1.4。
- 与打赏关系互补 vs 替代：`1/2`
理由：定位上符合 C1.2，但若内容继续退化为美图包，会迅速转向替代打赏。
- 内容形态稀缺性：`0/2`
理由：未满足 C1.5，从“美图包”转向“世界观档案+剧情连载+投票”尚未落地。
- 价格档位一致性：`0/2`
理由：多档价格、单档展示、无阶梯权益，用户心智不稳定。

综合判断：**2/10**。  
Phase 2 当前能支撑“首购尝试”，不能支撑“首月消费 + 次月续费”的完整闭环。若不补 P0 能力，这个方案更像一次性内容售卖系统，而不是符合 C1.2/C1.4/C1.5 的订阅系统。
