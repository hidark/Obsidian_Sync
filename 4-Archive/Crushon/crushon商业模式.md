## 二、畅聊包（Unlimited Chat Package）

> 本质：对 **Pro 模型**的无限使用权，用户级生效（与角色无关）。Ultra 模型不适用，正常按 credits 扣费。

### 商品列表

| 类型    | 有效期   | 价格      | 折扣      | 备注          |
| ----- | ----- | ------- | ------- | ----------- |
| 试用畅聊包 | 30 天  | $29.99  | 24% off | 新用户专属       |
| 月度畅聊包 | 30 天  | $39.99  | —       | 可单独购买或与订阅捆绑 |
| 年度畅聊包 | 365 天 | $299.99 | 37% off | 按年付，折算月均更低  |

### 使用规则

- Pro 模型 + 有效畅聊包 → 聊天零消耗额度，累计上限 40,000 条
- 支持与订阅**捆绑购买**（`New_Subscription_With_Unlimited_pkg`）
- 到期前 **7 天**自动推送提醒

## 三、消息额度（Message Credits）

> 订阅额度耗尽后的补充来源，也可单独购买使用。

### 消息扣费优先级

```
用户发消息
  ↓
① Pro 模型 + 有效畅聊包？→ 是 → 免费通行（上限 40,000 条）
  ↓ 否
② 订阅月度额度充足？→ 是 → 从订阅扣
  ↓ 否
③ 从钱包消息余额扣
  ↓
  余额不足 → 提示购买
```

## 五、打赏体系（Tips / Gift）

- 货币：钻石（Diamond）
- 触发：用户在聊天界面向角色发送礼物

## 七、系统能力边界

| 能力 | 状态 |
|------|------|
| 多渠道支付 | ✅ 十余个渠道，统一由 crushon-sub 处理回调 |
| 订阅升降级 | ✅ 完整支持，含月付 ↔ 年付 |
| 畅聊包与订阅解耦 | ✅ 可单独购买，也可捆绑 |
| 畅聊包作用范围 | ✅ 用户级 Pro 模型，非角色级 |
| 钻石打赏 | ✅ 支持礼物系统，匿名/实名均可 |
| 金币自由获取 | ✅ 多渠道免费获取 |
| 消息额度分层扣费 | ✅ 订阅 > 畅聊包(Pro) > 钱包，优先级清晰 |
| 价格/折扣修改 | ⚠️ 在 crushon-sub 侧，主服务透传，修改需跨服务协作 |
| 试用体系 | ✅ 订阅试用（SubscriptionStatusTrialing）+ 畅聊包试用包 |
